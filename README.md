# 🍌 Nupenca Bank — Seu banco sem frescura

O **Nupenca** é uma aplicação bancária moderna (Fintech) que combina uma estética premium com funcionalidades robustas de segurança e inteligência artificial. Com o slogan *"O banco que descasca pra você"*, o projeto foca em uma experiência de usuário (UX) fluida e visualmente impactante.

## ✨ Funcionalidades Principais

*   **🛡️ Modo Rua:** Proteção avançada que bloqueia transações e oculta o cartão virtual quando você está fora de casa.
*   **🤖 Nanica (IA Assistente):** Uma assistente virtual inteligente integrada que ajuda o usuário com dúvidas sobre saldo, Pix e educação financeira.
*   **💳 Cartão Premium 3D:** Um cartão virtual interativo e responsivo que gira e reage ao toque do usuário.
*   **🧺 Cofrinho das Bananas:** Sistema de metas de economia onde o usuário pode separar dinheiro do saldo principal.
*   **🔑 Pix & Pagamentos:** Transferências instantâneas com limites diários configuráveis.
*   **🌙 Alerta Noturno:** Monitoramento inteligente para transações realizadas entre 19h e 05h.

## 🛠️ Tecnologias Utilizadas

*   **Frontend:** HTML5, CSS3 (Vanilla) e JavaScript ES6+.
*   **Backend:** Google Firebase (Auth, Firestore, Hosting).
*   **IA:** Groq API (Llama 3).
